# Delivery Tracking PWA (Logística Real-time)

## Objetivo
Optimizar la trazabilidad de entregas y localización de flota para una PYME, eliminando la dependencia de servidores físicos mediante una arquitectura cloud serverless.

## Stack Tecnológico
- **Frontend:** Next.js 14 (App Router), Tailwind CSS.
- **Backend:** Supabase (PostgreSQL + Realtime).
- **IA/Visión:** Google Cloud Vision API para OCR de albaranes.
- **Mapas:** Leaflet / OpenStreetMap.

## Funcionalidades
- [ ] Escaneo de albaranes por foto (OCR).
- [ ] Seguimiento GPS cada 3 minutos (Optimización de batería).
- [ ] Visor público para comerciales sin registro.
- [ ] Borrado automático de datos tras 3 días (Privacidad y Eficiencia).
